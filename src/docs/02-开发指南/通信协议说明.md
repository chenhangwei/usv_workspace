# 通信协议说明

## 1. 话题 (Topics) 定义
### 1.1 PX4 交互 (uXRCE-DDS)
- **QoS**: Best Effort / Volatile (Sensor data)
- 关键话题:
  - `/fmu/in/trajectory_setpoint`: 控制指令
  - `/fmu/out/vehicle_odometry`: 里程计反馈

### 1.2 GS-USV 通信 (Zenoh Bridge)
- **QoS**: Reliable (Command/Ack) / Best Effort (High freq telemetry)
- 命名空间规则: `/usv_{id}/...`

## 2. 自定义消息结构
- `NavigationGoal.msg`: 包含位置、偏航角、速度限制。
- `UsvStatus.msg`: 包含电池、坐标、模式状态。
