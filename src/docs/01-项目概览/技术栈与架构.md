## 2. 核心技术栈
- **Middleware**: ROS 2 (Humble/Jazzy)
- **Flight Controller Firmware**: PX4 Autopilot (v1.14+)
- **Communication Bridge**: Zenoh / micro-XRCE-DDS
- **Language**: Python 3.10+

## 3. 关键模块说明
- **GS (Ground Station)**: 负责任务调度、状态监控、指令下发。
- **USV (Unmanned Surface Vehicle)**: 
  - Onboard Computer (OBC): 运行 ROS 2 驱动、控制节点。
  - Flight Controller (FC): 运行 PX4，负责底层姿态控制。
