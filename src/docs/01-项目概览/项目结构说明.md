# 项目结构说明

## 1. 目录结构概览
- `common_interfaces`: 自定义 ROS 消息 (msg) 和服务 (srv) 定义。
- `common_utils`: 通用 Python 工具库（参数加载、数学计算等）。
- `gs_bringup`: 地面站启动文件与配置。
- `gs_gui`: 地面站图形化界面与核心逻辑节点。
- `usv_control`: USV 端的控制逻辑（对接 PX4 Offboard 模式）。
- `usv_drivers`: 传感器驱动（如 UWB 驱动）。
- `usv_comm`: 通信与状态聚合节点。
